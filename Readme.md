📝 # Task List Manager
A command-line manager to create, view, update, and delete task lists and their tasks.

✨ # Features
Create and manage task lists: Organize your tasks into different lists.

Add, delete, and mark as completed: Full control over the tasks within each list.

Data persistence: Data is saved to a local file, so you don't lose your progress when you exit the application.

🚀 # Getting Started
To run this project, you need to have Java installed.

Clone this repository to your local machine.

Compile and run the MainTaskList.java class.

Use the menu options to interact with the task manager.
